how to run this project:

after cloning project

```shell
#configure .env variables
VITE_COOL_API - base backend url
VITE_COOL_URL - base domain

#run
npm install
npm run dev
#project will run on server

#production build
npm run build

#error checking
npx eslint .

#prettier
npm run format


```
